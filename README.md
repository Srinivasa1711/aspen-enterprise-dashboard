# aspen-enterprise-dashboard

This project demonstrates a simple enterprise dashboard system inspired by my professional experience at Aspen Technologies.

## Overview
The system consists of a Python Flask backend and a web-based frontend that displays employee performance data.

## Features
- REST API built using Flask
- Frontend dashboard using HTML, CSS, JavaScript
- API integration between backend and frontend

## Architecture
- Backend: Python Flask API
- Frontend: HTML, JavaScript
- Data: Mock dataset for demonstration

## How to Run
1. Navigate to the backend folder
2. Run: python app.py
3. Open frontend/index.html in the browser

## Key Learning
- API development using Flask
- Frontend-backend integration
- Structuring enterprise applications
